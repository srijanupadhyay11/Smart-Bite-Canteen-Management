🍽️ Smart Bite – Smart Canteen Management System
📌 Overview

Smart Bite is a smart canteen management system designed to simplify food ordering and reduce waiting time. It provides a seamless digital experience where users can browse menus, place orders, and track them in real-time, while canteen staff can efficiently manage incoming requests.

🚀 Features
📱 Digital Menu – Browse food items with prices and availability
🛒 Cart System – Add/remove items easily before ordering
💳 Quick Payments – Smooth and fast checkout process
📊 Real-Time Order Tracking – Track order status instantly
📺 Smart Display (IoT) – Displays order numbers and status
⚡ Fast & Efficient – Reduces queue and waiting time
